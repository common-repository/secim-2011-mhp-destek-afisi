== Description ==



Bu eklenti Wordpress blogunuzda 2011 seçimleri için Milliyetçi Hareket Partisi (MHP) destekleyen blog yazarları için yan menüde bileşen olarak resmi seçim afişi gösterilmesini ve MHP'nin resmi sitesine gidilmesini sağlıyor.



Bir örneğini <a href="http://www.mentalmasturbasyon.com">Mental Masturbasyon</a> sitesinin yan menüsünde görebilirsiniz.







== Installation ==



1. Wordpress Admin panelinde, EKLENTİLER > Yeni EKLE ve  'Seçim 2011' olarak arama yapın.

2. 'Yükle' Tıklayın.

3. Wordpress Admin içinde , Görünüm > Bileşenler, içinde  'Seçim 2011 MHP Afişi' istediğiniz bölüme sürükleyip, yerleştirin ve kaydedin..
